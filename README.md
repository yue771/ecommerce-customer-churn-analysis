# E-commerce Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer behavior in an e-commerce dataset and builds machine learning models to predict customer churn.

## 📊 Key Analysis
- Customer segmentation based on spending
- Discount impact on purchasing behavior
- Customer satisfaction analysis
- Recency vs churn relationship

## 🤖 Modeling
- Logistic Regression
- Random Forest (with GridSearchCV tuning)

## 📈 Evaluation
- Accuracy & Classification Report
- ROC Curve (AUC)
- Confusion Matrix
- Feature Importance Analysis

## 🔍 Key Findings
- Discount usage and total spending are the most influential features
- Customer satisfaction also impacts churn behavior

## ⚠️ Limitations
- The dataset is synthetic
- Churn label is rule-based, so model performance may be overly optimistic

## 🛠 Tech Stack
- Python
- Pandas / NumPy
- Scikit-learn
- Matplotlib
